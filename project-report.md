## 项目概述

本项目是一个以个人品牌为核心的静态个人网站，旨在以非简历的方式向外界呈现自己——不只展示技能与经历，更展现思维方式、多元身份与个人气质。网站作为求职作品集的补充材料，供面试官在正式简历之外进一步了解候选人。

---

## 技术实现

- **技术栈**：纯 HTML5 + CSS3 + 原生 JavaScript，无框架依赖
- **字体**：Playfair Display（衬线标题）+ Jost（无衬线正文），通过 Google Fonts 引入
- **部署**：GitHub 托管源码，Vercel 自动化部署，支持 HTTPS

主要实现的技术细节包括：

- Canvas API 绘制 Hero 区光线流动动效（shimmer streaks）
- CSS `backdrop-filter` 实现毛玻璃面板效果
- Intersection Observer API 实现滚动触发的渐入动画
- CSS `radial-gradient` + `animation` 实现各 Section 的呼吸感背景光晕
- Ken Burns 效果（`animation` + `transform: scale`）用于首页背景照片
- 响应式布局，适配移动端与桌面端

## 页面结构

### 主页 `index.html`

共五个 Section，采用渐进式叙事结构：

| Section | 内容 |
|---|---|
| Hero | 全屏背景照片 + 动态光效 + 身份标签云 |
| About me | 个人介绍 + 倾斜艺术照 + 信条引言 |
| Identities | 六张身份卡片，两张带子页面入口 |
| Vibes | 兴趣爱好展示，含 ENFP 官方性格描述 |
| Now | 当下状态记录（在读、在看、在想） |

### 塔罗师页 `tarot.html`

全屏塔罗牌照片作为 Hero 背景，毛玻璃数据浮卡展示经验数据，正文分「缘起」「收获」「理念」三个板块，采用图文嵌入式排版。

### 志愿者页 `volunteer.html`

志愿课堂照片作为 Hero 背景（已做模糊处理，标注保护孩童隐私），正文分「团队故事」「课程设计」「个人收获」三个板块，图片采用偏移装饰边框的嵌入式卡片样式。

## 设计思路

整体风格定位为**轻成熟、柔和、有高级感**，具体体现在：

- 配色以奶油米白（`#faf7f2`）为主色调，琥珀暖色（`#c4956a`）、薄荷绿（`#7aaa82`）、淡紫（`#9b94c0`）作为各身份维度的点缀色
- 排版强调留白与呼吸感，标题与正文字重形成明显层次
- 动效克制，以渐入、呼吸、光效为主，避免过度打扰阅读
- 照片处理（倾斜艺术照、全屏背景、模糊隐私处理）增强视觉记忆点

- 个人兴趣（ENFP、R&B 音乐、电影、旅行、二次元）
